## 📌 Overview  
The Rover Robotic Arm is a multifunctional robotic system that integrates a mobile rover with a robotic arm. It is designed for tasks such as object handling, navigation, and remote operation in hazardous or inaccessible environments.  

This project combines mechanical design, embedded systems, and wireless communication.

## 🎯 Objectives  
- Design a rover-based robotic system  
- Build a multi-degree robotic arm  
- Enable wireless control using ESP32  
- Perform pick-and-place operations  

---

## 🧠 Working Principle
User Input → ESP32 → Motor Driver → Motors & Servos → Action

- ESP32 receives commands via Wi-Fi  
- Motor driver controls rover movement  
- Servo motors control robotic arm joints  
- System performs real-time actions  

---
